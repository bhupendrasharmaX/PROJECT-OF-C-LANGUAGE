# 🖥️ C Language Projects

A collection of **8 real-world mini projects** built in C, covering core programming concepts like file handling, dynamic memory, data structures, and user interaction.

---

## 📂 Projects Overview

| # | Project | Description |
|---|---------|-------------|
| 01 | 🔢 Simple Calculator | Performs basic arithmetic operations (+, -, *, /) |
| 02 | 🎓 Student Marks Management System | Add, display, and manage student marks |
| 03 | 🎮 Number Guessing Game | A fun game where the user guesses a random number |
| 04 | 📚 Library Management System | Manage books – add, search, issue, and return |
| 05 | 🏦 Bank Management System | Account creation, deposit, withdrawal, and balance check |
| 06 | 🗜️ File Compression | Compress and decompress files using C |
| 07 | 🗂️ Dynamic Student Record System | Manage student records using dynamic memory allocation |
| 08 | 📦 Mini Inventory Management System | Track products, stock, and inventory in a store |

---

## 🛠️ How to Compile & Run

Make sure **GCC** is installed on your system.
```bash
# Clone the repository
git clone https://github.com/bhupendrasharmaX/PROJECT-OF-C-LANGUAGE.git

# Navigate into the folder
cd PROJECT-OF-C-LANGUAGE

# Compile any project
gcc 01-Simple_Calculator.c -o calculator

# Run it
./calculator
```

> On Windows, use `calculator.exe` instead of `./calculator`
